# Docs Coverage Bot Support

This repository is the public issue tracker for Docs Coverage Bot.

Use it to report:
- installation problems
- incorrect coverage reports
- billing problems
- feature requests

Before opening an issue:
- do not include secrets, tokens, private keys, or private repository contents
- include the repository type, language, and the relevant check output when possible
